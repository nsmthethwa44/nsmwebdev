Behind The Code

Who I am, the stack I use, and the kind of applications I build.

I’m a Full-Stack Software Developer working with .NET and Angular to build practical, production-ready web applications. My focus is clean architecture, maintainable code, and systems that solve real business problems.

I design REST APIs with ASP.NET Core, build responsive interfaces with Angular, and work across SQL Server or PostgreSQL databases. My experience includes booking platforms, auction systems, real estate management tools, and other workflow-driven applications.

How I Build

How I approach systems, architecture, and long-term maintainability.

I build applications as structured systems, not disconnected features. My priority is clear separation between layers, predictable data flow, and business logic that lives in the service layer instead of leaking into controllers or the UI.

This approach results in software that’s easier to extend, debug, and maintain over time without fragile shortcuts, hidden side effects, or tightly coupled components.
