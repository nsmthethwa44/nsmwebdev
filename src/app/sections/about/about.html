Behind The Code

Who I am, the stack I use, and the kind of applications I build.

I’m a Full-Stack .NET & Angular Developer working with .NET and Angular to build practical, production-ready web applications. My focus is clean architecture, maintainable code, and systems that solve real business problems.

I design REST APIs with ASP.NET Core, build responsive interfaces with Angular, and work across SQL Server or PostgreSQL databases. My experience includes booking platforms, auction systems, real estate management tools, and other workflow-driven applications.

How I Build Systems

How I approach systems, architecture, and long-term maintainability.

I design applications as cohesive systems, not disconnected features. I use clean, layered architecture with clear separation of concerns, predictable data flow, and business logic handled in the service layer.

This results in secure, maintainable software that’s easy to extend, debug, and scale without fragile shortcuts or tightly coupled components.
